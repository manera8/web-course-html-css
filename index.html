🎬 Начать обучение

📄 Что такое HTML?

HTML — это язык разметки, который создаёт структуру веб-страницы: заголовки, абзацы, изображения, ссылки.

🎨 Что такое CSS?

CSS отвечает за стиль: цвета, шрифты, расположение блоков. Делает сайт красивым и удобным.

🛠 Что вы научитесь делать?

Верстать макеты, создавать меню, карточки, формы и адаптивные страницы под телефон и ПК.

🎥 Видео-уроки по вёрстке

📌 Урок 1: Основы HTML

Изучим теги: заголовки, параграфы, списки, изображения и ссылки.

🎨 Урок 2: Основы CSS

Селекторы, цвета, шрифты, рамки. Превращаем скучный текст в стильный дизайн.

📱 Урок 3: Flexbox и адаптивность

Создаём гибкие макеты, которые красиво смотрятся на любом экране.

💡 Совет: после каждого видео открывай редактор кода и повторяй примеры!

💻 Живой пример кода

Создаём карточку товара

<div class="card">
  <h3>Ноутбук Pro</h3>
  <p>Мощный процессор, 16GB RAM</p>
  <span class="price">₾4999</span>
  <button>Купить</button>
</div>

✨ Этот код создаёт карточку товара. CSS добавляет тени, скругления и красивый шрифт.
